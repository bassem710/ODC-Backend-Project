## APIs 
### Admins
**GET** /api/admins/my-data \
**GET** /api/admins/ \
**GET** /api/admins/:id \
**POST** /api/admins/ \
**POST** /api/admins/login \
**PATCH** /api/admins/:id \
**DEL** /api/admins/:id

### Courses
**GET** /api/courses/ \
**GET** /api/courses/:id \
**GET** /api/courses/money \
**GET** /api/courses/recommended/:id \
**POST** /api/courses/ \
**PATCH** /api/courses/:id \
**DEL** /api/courses/:id 

### Students
**GET** /api/students/ \
**GET** /api/students/:id \
**GET** /api/students/top \
